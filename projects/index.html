A RC Car Robot with 4 Mecanum Wheels Controlled by A PS2 Joystick

The robot is an RC Car using an Arduino and a PS2 controller with 4 mecanum wheels and a gripper for picking up coins. The coins are square-shaped cork. The robot's task is to pick up the coins and place them in the designated location according to the pilot holding the PS2 controller. This robot is used in the Indonesian Robot Contest (Kontes Robot Indonesia- KRI) 2022 in the Indonesian Thematic Robot Contest (Kontes Robot Tematik Indonesia - KRTMI) category. The robot can pick up coins vertically and place them horizontally.

Visit Indonesia, A Website for Selling Tourism Tickets in Purbalingga

Visit Indonesia is a website for selling tourism tickets and products as well as culinary items in Purbalingga. The website features a payment gateway through Midtrans. The website was created by a team of 4 people, with my focus being on designing and creating the database as well as developing the backend of the website using Laravel framework. To check out Visit Indonesia, you can visit the following link : https://visit-indonesia.id

Prediction of Diabetes Using Artificial Neural Network Classification Algorithm

Prediction of the likelihood of having Diabetes using Artificial Neural Network (ANN) classification algorithm. This prediction uses a reference of 80,000 classified data records with diabetes as a parameter including age, gender, body mass index (BMI), hypertension, heart disease, smoking history, HbA1c level, and blood glucose level. This dataset can be used to build a machine learning model to predict diabetes in patients based on their medical history and demographic information. In this data mining process, diabetes prediction is performed on 20,000 data, which in this case refers to patients.
